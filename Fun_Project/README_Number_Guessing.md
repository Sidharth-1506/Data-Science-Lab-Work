Project 2: Number Guessing Game

Description

This project is an interactive game where the computer generates a random number, and the user has to guess it within a limited number of attempts. The objective is to practice loops, conditions, and randomness in Python.

Dataset

Not applicable (random number generation using Python)

Steps Performed

Generated a random number using the random library

Set a fixed number of attempts

Used a loop to allow multiple guesses

Provided feedback (Too High / Too Low)

Ended the game when the user guesses correctly or runs out of attempts

Results

Interactive gameplay experience

Demonstrates control flow using loops and conditions

Provides real-time feedback to the user

Tools Used

Python

random library

Built-in functions: input(), print()

Control structures: if-else, while loop

Conclusion

This project strengthened understanding of loops, conditional logic, and randomness. It also introduced basic game logic and user interaction in Python.
